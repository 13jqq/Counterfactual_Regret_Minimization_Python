### Introduction
This repository contains the example code of regret minimiaztion solving Rock-Paper-Scissors and Kuhn poker problems.

* Rock-Paper-Scissors (RPS)
    * Self training
    * Finding RPS Equilibrium
* Kuhn poker
    * Self training
